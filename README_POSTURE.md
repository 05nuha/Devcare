# Posture Detector

Distance-aware, calibrated posture detection using MediaPipe and OpenCV.

## Features
- Distance-invariant (works at any distance)
- Personalized calibration
- Robust proportional measurements
- Real-time scoring
- Visual test mode

pip install opencv-python==4.8.1.78
pip install mediapipe==0.10.8
